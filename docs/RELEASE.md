# Viking Chronology — release notes

## 1.0.0

Первая production-публикация интерактивного исторического 3D-атласа.

### Надёжность

- мобильная сборка ES2018;
- автономная IIFE-сборка без module script;
- WebGL diagnostics и context recovery;
- стабильные GPU-профили `high`, `balanced`, `battery`.

### Взаимодействие

- автовращение полностью удалено;
- увеличены области попадания исторических маркеров;
- tap/click отделён от жеста вращения камеры;
- информационная карточка закрывается только явным действием.

### Визуализация и звук

- плавные Catmull-Rom маршруты и quaternion-ориентация кораблей;
- исторически дисциплинированная средневековая дизайн-система;
- процедурная звуковая реконструкция моря, ветра, дерева и инструментальных тембров.
