# Performance budget

| Метрика | Целевое значение |
|---|---:|
| Начальный JS без Three.js, gzip | < 100 KB |
| Three.js chunk, gzip | < 300 KB |
| CSS, gzip | < 10 KB |
| Standalone HTML | < 1.7 MB |
| Mobile DPR, auto | 0.82–1.3 |
| Desktop DPR, auto | до 1.8 |
| Активные корабли | не более 3 |
| Mobile timeline React commits | ≤ 12/сек |

## Устранённые источники рывков

- переход корабля между концом и началом сегмента теперь скрыт плавным envelope;
- ориентация строится по касательной и радиальной нормали, затем сглаживается quaternion slerp;
- путь интерполируется `CatmullRomCurve3`, а не линейно между 64 точками;
- большие `delta` после сворачивания вкладки ограничиваются;
- DPR не меняется постоянно в зависимости от кратковременного FPS.
