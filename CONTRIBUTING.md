# Вклад в проект

1. Создайте отдельную ветку.
2. Не добавляйте исторические утверждения без институционального или академического источника.
3. Выполните `npm install` и `npm run check`.
4. Для изменений рендера укажите устройство, браузер, средний FPS и профиль качества.
5. Не добавляйте внешние аудиофайлы без документированной лицензии.
