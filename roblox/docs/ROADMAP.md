# Production Roadmap

## Phase 0 — Foundation (delivered here)

Architecture, networking, profiles, streaming configuration, expedition state, camera/UI/audio controllers, validation and documentation.

## Phase 1 — Scandinavia vertical slice

- final fjord terrain;
- production settlement kit;
- one complete longship;
- 12 named characters;
- one expedition chapter;
- first professional audio and animation pass;
- low-end mobile certification.

## Phase 2 — North Sea and Britain

Weather, sea navigation, ports, faction reputation, trade, conflict and first multiplayer expedition systems.

## Phase 3 — Baltic and eastern rivers

Portage mechanics, river vessel variation, trade networks, diplomacy and multi-Place travel.

## Phase 4 — Atlantic frontier

Iceland, Greenland and western archaeological chapter with stricter evidence gating and survival systems.

## Staffing reality

A large polished experience requires environment art, character art, animation, technical art, gameplay/network engineering, UI/UX, audio, production, QA and historical/linguistic review. The code foundation reduces risk; it does not replace that production team.
